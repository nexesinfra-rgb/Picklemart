# Pickle Mart

Pickle Mart is an e-commerce Flutter application specializing in authentic homemade pickles, karam podis (spice powders), and traditional masalas. The app features both customer-facing and admin interfaces with support for multiple platforms (mobile, tablet, desktop).

## Features

- Browse authentic pickles, karam podis, spice powders, and masalas
- Shopping cart and order management
- Delivery address selection with map picker
- User account management
- Admin panel for product and order management
- Real-time updates with Supabase backend

## Getting Started

This project is built with Flutter and uses Supabase as the backend.

For help getting started with Flutter development, view the
[online documentation](https://docs.flutter.dev/), which offers tutorials,
samples, guidance on mobile development, and a full API reference.
